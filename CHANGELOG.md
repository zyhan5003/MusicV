# 更新日志

本文档记录 MusicV 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增
- 添加麦克风实时输入模式，支持实时音频可视化
- 新增独立特效库架构（effects_origin/）
- 新增雨特效（RainEffect）：音频振幅驱动的雨滴效果
- 新增雪特效（SnowEffect）：音频节拍驱动的雪花飘落效果
- 新增火特效（FireEffect）：音频强度驱动的火焰效果
- 新增花瓣特效（PetalEffect）：音频节奏驱动的花瓣飘落效果
- 新增闪光方块特效（GlowingSquaresEffect）：音频特征驱动的方块动画效果
- 波形可视化支持镜像显示，上下对称渲染
- 音频特征提取支持起音包络（onset_envelope）
- 添加数据预览窗口，实时显示音频特征

### 优化
- 优化粒子生成逻辑，基于音频强度动态调整
- 提高麦克风模式响应速度（采样率 16000Hz，更新率 60 FPS）
- 优化方块特效动画，减少视觉闪烁
- 改进音频特征提取性能

### 修复
- 修复多次播放时窗口不显示的问题
- 修复停止可视化时窗体残留问题
- 修复 ESC 键导致系统崩溃的问题
- 修复麦克风模式下可视化不联动的问题
- 修复窗体关闭按钮卡住系统的问题

### 文档
- 新增中英文双语文档（README.md 和 README_EN.md）
- 新增贡献指南（CONTRIBUTING.md 和 CONTRIBUTING_EN.md）
- 新增 MIT 许可证文件（LICENSE）

## [0.1.0] - 2024-01-01

### 新增
- 初始版本发布
- 支持主流音频格式（MP3/WAV/FLAC）解析
- 实现多维度音频特征提取（时域、频域、节奏、音色）
- 实现 2D 可视化（波形、频谱、均衡器）
- 实现 3D 可视化（频谱立方体、3D 模型）
- 实现粒子系统（基础粒子、节拍粒子、跳跃粒子）
- 实现 GUI 界面（文件选择、可视化类型切换、参数配置）
- 支持配置文件管理（YAML）
- 支持数据导出功能
